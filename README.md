#  MY Habit Heatmap Tracker

A full-stack habit tracking application with GitHub-style contribution graph, user authentication, and MongoDB persistence.

## ✨ Features

- 🔐 **User Authentication**: Signup/Login with email and password
- 📊 **Visual Heatmap**: 365-day tracking grid
- 🔥 **Streak Counter**: Track your consistency
- 📈 **Weekly Statistics**: Monitor progress
- 🌙 **Dark Mode**: Comfortable viewing day/night
- 🔒 **Secure Deletion**: Password-protected habit deletion
- 💾 **MongoDB Storage**: Persistent data across devices
- 📱 **Responsive Design**: Works on all devices

## 🛠️ Tech Stack

### Frontend
- React 18
- React Router DOM
- Axios
- CSS3 with Custom Properties

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs for password hashing
